To run this test you need following:
Selenium Grid
!note java -jar selenium-server-standalone-2.35.0.jar -role hub
Selenium Node with support for required driver (default config)
!note java -jar selenium-server-standalone-2.35.0.jar -role node -host %COMPUTERNAME% -hub !-http://localhost:4444/grid/register-! 
or specific config:
!note c:\tools\jre\bin\java -jar selenium-server-standalone-2.35.0.jar -role node -host %COMPUTERNAME% -hub !-http://localhost:4444/grid/register-! -nodeConfig win.json
